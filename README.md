## Description

Car journal, car and driver storage system. One machine can be assigned to several machines.
Functionality:
- Add Car
- Add driver
- Assign car to driver
- Search car, output: car info + drivers list assigned to car

## Installation

# Docker
```bash
$ docker-compose up -d
```
# Nest.JS CLI
```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
